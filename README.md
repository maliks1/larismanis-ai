## Features

- Voice-ledger parsing with Gemini and Supabase inserts.
- Realtime dashboard metrics for Current Ratio and Net Profit Margin.
- Marketing generator with real image upload to Supabase Storage.

## Environment Variables

Create a `.env.local` with at least these values:

```bash
GOOGLE_GENERATIVE_AI_API_KEY=your_google_ai_key
GEMINI_MODEL=gemini-2.5-flash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Optional but recommended for storage uploads:

```bash
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

## Supabase Setup

- Enable Realtime on the `transactions` table.
- Create a public Storage bucket named `marketing-assets`.
- Make sure the `transactions` schema matches `src/types/database.types.ts`.
